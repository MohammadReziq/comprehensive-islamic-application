# التطبيق — التدفقات الأساسية (باختصار)

## 1. تسجيل كإمام (Imam)

- **التسجيل:** تختار "إمام / مشرف" وتنشئ الحساب.
- **بعد التسجيل:** التطبيق يوديك على **بوابة المسجد**.
- هناك: **إنشاء مسجد جديد** أو **الانضمام بكوْد**. عندما تنشئ مسجد = أنت أرسلت **طلب لمدير النظام (Super Admin)**.
- **حالة "طلبك قيد المراجعة":** تبقى على هذه الشاشة حتى يقبل أو يرفض مدير النظام طلبك. عندما يقبل → تنتقل تلقائياً إلى لوحة المشرف (تحضير، طلاب، إلخ).

**باختصار:** إمام = يطلب (ينشئ مسجد) ← ينتظر موافقة السوبر أدمن.

---

## 2. دخول كمدير نظام (Super Admin)

- **تسجيل الدخول** بحساب مدير النظام.
- التطبيق يوديك مباشرة على **صفحة طلبات المساجد** (طلبات الأئمة).
- تعرض القائمة: كل الطلبات قيد المراجعة. عندك زر **موافقة** وزر **رفض** لكل طلب.
- عندما توافق على مسجد ← الإمام صاحب الطلب يصير يرى لوحة المشرف ومسجده معتمد.

**باختصار:** سوبر أدمن = يشوف الطلبات ويوافق أو يرفض.

---

## 3. دخول كولي أمر (Parent / Wali Al-Amar)

- **تسجيل الدخول** بحساب ولي الأمر.
- التطبيق يوديك على **الصفحة الرئيسية لولي الأمر** (الرئيسية).
- منها: **أطفالي**، إضافة أبناء، ربطهم بمسجد، ومتابعة حضورهم ومعلوماتهم (نقاط، بطاقة QR، إلخ).

**باختصار:** ولي الأمر = صفحته الخاصة: يضيف أولاده ويشوف معلوماتهم.

---

## ملخص التوجيه

| الدور        | بعد تسجيل الدخول أو التسجيل → يروح إلى |
|-------------|----------------------------------------|
| **إمام**    | بوابة المسجد (إنشاء/انضمام ثم انتظار الموافقة أو لوحة المشرف) |
| **مدير نظام** | صفحة طلبات المساجد (موافقة/رفض) |
| **ولي أمر** | الصفحة الرئيسية (أطفالي ومعلوماتهم) |
